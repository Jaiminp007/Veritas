## What This Project Is

Senso Veritas is an independent benchmark that runs 25 adversarial financial compliance queries through two LLMs in parallel вҖ” a raw baseline (llama3.1:8b via Ollama) and a Senso-grounded LLM вҖ” then scores both using an external judge and computes a Narrative Control Score (NCS) for each. The delta between baseline NCS and Senso NCS is the headline metric that proves Senso's ground-truth layer reduces hallucinations.

**This is NOT a chatbot. It is a benchmark pipeline.**

The pipeline runs locally. Results are persisted to Convex. The dashboard is a static React app deployed to Vercel that reads from Convex. No backend is deployed.

## NCS Formula вҖ” Canonical

```
raw_NCS  = (Оұ Г— CitationMatch) вҲ’ (ОІ Г— HallucinationPenalty) + (Оҙ Г— KeyFactCoverage)
norm_NCS = (raw_NCS + ОІ) / (Оұ + ОІ + Оҙ)    вҶ’  clamped to [0.0, 1.0]

Default weights:  Оұ=0.5  ОІ=0.8  Оҙ=0.3
Range proof:
  Best  raw =  0.5(1) вҲ’ 0.8(0) + 0.3(1) =  0.8  вҶ’  norm = 1.0
  Worst raw =  0.5(0) вҲ’ 0.8(1) + 0.3(0) = вҲ’0.8  вҶ’  norm = 0.0
```

**Compute for BOTH baseline and Senso. Delta = senso_norm - baseline_norm.**

Baseline citation_match is always 0.0 (no documents). Baseline key_fact_coverage uses the same function.

## How to Run

### Prerequisites
```bash
# 1. Start Ollama with correct models
ollama serve
ollama pull llama3.1:8b
ollama pull qwen2.5:7b

# 2. Verify Ollama is working
curl http://localhost:11434/api/tags

# 3. Install Python deps
pip install -r requirements.txt

# 4. Set up env
cp .env.example .env.local
# Fill in SENSO_API_KEY and CONVEX_DEPLOYMENT
```

### Run pipeline
```bash
# Start FastAPI
uvicorn app.main:app --reload --port 8000

# In another terminal вҖ” trigger pipeline
curl -X POST http://localhost:8000/pipeline/run

# Check status
curl http://localhost:8000/pipeline/status/{job_id}

# Check metrics
curl http://localhost:8000/metrics/summary
```

### Deploy frontend
```bash
cd frontend
npm install
npx convex deploy    # Push schema changes to Convex first
vercel --prod        # Deploy React to Vercel
```

## What Success Looks Like

After all bugs are fixed and a full 25-query run completes:

| Metric | Expected Range |
|---|---|
| Baseline avg NCS | 15% вҖ“ 35% |
| Senso avg NCS | 40% вҖ“ 65% |
| NCS Delta (Senso advantage) | +15% вҖ“ +30% |
| Baseline hallucination rate | 65% вҖ“ 80% |
| Senso hallucination rate | 25% вҖ“ 45% |
| Senso refusal rate | 10% вҖ“ 25% (not hallucinations) |

If Senso NCS < Baseline NCS after fixes, that is a valid and interesting finding вҖ” present it honestly.

## Convex Mutation вҖ” writeRecord

```typescript
// convex/audit.ts
import { mutation } from "./_generated/server";
import { v } from "convex/values";

export const writeRecord = mutation({
  args: {
    trace_id: v.string(),
    query_id: v.string(),
    category: v.string(),
    query_text: v.string(),
    ground_truth_answer: v.string(),
    key_facts: v.array(v.string()),
    source_document: v.string(),
    baseline_response: v.string(),
    baseline_hallucination_detected: v.boolean(),
    baseline_hallucination_reason: v.string(),
    baseline_hallucination_penalty: v.float64(),
    baseline_ncs: v.float64(),
    baseline_raw_ncs: v.float64(),
    senso_response: v.string(),
    senso_citations: v.array(v.string()),
    senso_hallucination_detected: v.boolean(),
    senso_hallucination_reason: v.string(),
    senso_hallucination_penalty: v.float64(),
    narrative_control_score: v.float64(),
    raw_ncs: v.float64(),
    ncs_delta: v.float64(),
    citation_match: v.float64(),
    key_fact_coverage: v.float64(),
    weights_used: v.object({ alpha: v.float64(), beta: v.float64(), delta: v.float64() }),
    latency_ms: v.float64(),
    status: v.union(v.literal("success"), v.literal("failed")),
    error_message: v.optional(v.string()),
    created_at: v.int64(),
  },
  handler: async ({ db }, args) => {
    return await db.insert("auditRecords", args);
  },
});
```
